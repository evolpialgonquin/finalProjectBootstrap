HTML 5

HTML - Hypertext Markup Language is used in my work to create the structure of web pages throughout the manipulation and proper formatting of text, image, hyperlinks and other elements, organized within grids, flexboxes and/or one of the various tabular components of this amazing language.

CSS 3

CSS - Cascading Style Sheets is applied to formatting the style and structure of my web pages. The language interacts with the HTML throughout the targeting of its elements and gives the page's colors, layouts, stilized fonts, image and videos, amongst many others aesthetic and organizational elements.

Bootstrap 4

Bootstrap is a framework used in my projects to implement custom designs for typography, forms, buttons, navigation bars, and also helps with accessibility tools and components to the web pages. All of the work is made through direct interactions with HTML and CSS.

SASS

SASS - Syntactically Awesome Style Sheets is a CSS prepocessor used in my projects to optimize the use of everything CSS can offer. Yet relativelly knew to me, I'm learning to explore the most of this language's features with the use of, e.g., variables and nested rules.

JavaScript

JavaScript is used in my projects to implement complex features and user interaction components to the web pages. This language interacts directly with HTML and sometimes with CSS.

jQuery

jQuery is seen by me as a "super-library" for JavaScript. It helps immensely my work with easier and faster HTML/DOM and CSS manipulation, event handlers, effects and animations and other utilities.

Python

Python was mostly used by me for object oriented programming so far, nevertheless the use of libraries and tools that allow complex interactions with databases, web browsers, HTML and indirectly JavaScript and CSS is helping me to develop my knowledge in the vast and beautiful world of back-end programming.

Flask

Flask is a Python web framework used in my projects to allow communications between Python and a web browser with its respective source code.

MySQL

MySQL is used by me for creating and managing/manipulating simple databases. Still in the proccess of learning more complex uses for this system.

SQLite

SQLite is another tool that provides a relational database management system. It is mostly used inside of Python programming.
